# 血糖・食事・インスリン記録ノート

血糖値・食事（炭水化物・食物繊維）・インスリン単位を記録し、
**公知の計算式による集計**を主治医との相談用に整理する、スマホ用の Web/PWA アプリです。

**アプリ（公開版）: https://carb-icr.azaslab.app/**

- 外部ライブラリなし・通信なし（データは端末内の IndexedDB にのみ保存）
- ブラウザで開くだけで動作。ホーム画面に追加すればアプリのように使えます
- 炭水化物・食物繊維・血糖値は**手入力**します

> **本アプリが行わないこと**
> 本アプリは、インスリン投与量の計算・提示、カーボ比（ICR）やインスリン効果値（ISF）など
> 治療パラメータの推奨・設定への適用、診断・治療に関する判断を**一切行いません**。
> 表示されるのは、記録した値と、公知の計算式・一般的な統計処理（合計・中央値）による集計のみです。
> 集計結果は、受診時に主治医にお見せいただくための資料です。

## 使い方

### 1. 記録（毎食）
- 朝/昼/夕/間食 を選ぶ
- **炭水化物・食物繊維** → 糖質（炭水化物 − 食物繊維）を自動表示
- **インスリン単位**、**食前血糖・食後血糖**（食後は2〜3時間後のピーク後）を入力
- 「記録する」

### 2. 集計
記録がたまると、各食事の「糖質 ÷ インスリン単位」（公知のカーボカウント法の基本式）と、
その中央値（全体・時間帯別）を表示します。

- 中央値の集計対象は、**食前後の血糖差が ±30 mg/dL 以内**に収まった食事のみです
  （「血糖がほぼ食前値に戻った食事で比を確認する」という、カーボカウント指導の文献に
  記載のある検証手順に基づきます。下記「集計方法と出典」参照）
- 1日インスリン量（TDD）は、基礎＋ボーラスの**単純合計**（ポンプ実測値があれば優先）です
- 「主治医用レポート」ボタンで、直近の記録と集計を印刷用に整形できます

> ⚠ **医療上の注意**: 集計値は記録を整理した参考値です。本アプリは投与量や治療パラメータを
> 決定しません。投与量・治療方針の調整は必ず主治医の指示に従ってください。

### 3. 設定
- 基礎インスリン1日量（TDDの単純合計にのみ使用）
- インスリン方式の期間登録（記録の分類用）
- バックアップ（JSON）の書き出し・読み込み（機種変更前に書き出してください）

## 集計方法と出典

本アプリの計算はすべて公知の式・一般的な統計処理であり、独自のアルゴリズムによる
指標の算出は行いません。

| 計算 | 式・方法 | 位置づけ・出典 |
|---|---|---|
| 糖質 | 炭水化物 − 食物繊維 | 公知の栄養計算（いわゆるネットカーボ） |
| 糖質/インスリン比 | 糖質(g) ÷ インスリン(単位) | カーボカウント法における糖質/インスリン比（ICR）の基本定義。日本糖尿病学会 編・著『カーボカウントの手びき ―「糖尿病食事療法のための食品交換表」準拠―』文光堂, 2017（ISBN 978-4-8306-6064-1）／医療者向け『医療者のためのカーボカウント指導テキスト』文光堂（ISBN 978-4-8306-6063-4）。 |
| 集計の採用基準 | 食前後の血糖差 ±30 mg/dL 以内の食事のみ | インスリン/カーボ比の検証手順（食後に血糖がほぼ食前値±30 mg/dL へ戻るかで比の妥当性を確認する）。John Walsh & Ruth Roberts, *Using Insulin: Everything You Need for Success with Insulin*, Torrey Pines Press（ISBN 978-1-884804-85-4）。 |
| 代表値 | 中央値（全体・時間帯別） | 一般的な統計処理 |
| TDD | 基礎 + ボーラスの単純合計（実測値優先） | 単純合計 |

出典の方針: 本アプリが実装する計算は、いずれも公知の式・確立した手順・一般的な統計処理に
限り、出典を上表に明示できるものだけを採用します。出典を確定できない計算は実装しません。
（上表の※印は、書誌は特定済みで、該当ページのみ版の入手後に追補する項目です。）
